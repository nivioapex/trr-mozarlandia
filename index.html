📋 Imóveis CAR individuais — dados não importados ainda
A tabela car_propriedades está criada mas vazia. Para ver os 24.081 imóveis como marcadores individuais:

1. Acesse consultapublica.car.gov.br · informe e-mail e resolva o captcha por município
2. Baixe o shapefile (.zip) de cada município
3. Converta para CSV com lat/lon (centroide) usando QGIS ou Python/geopandas
4. Importe para a tabela car_propriedades no Supabase (colunas: cod_imovel, nom_imovel, municipio, uf, area_ha, lat, lon)

⚠️ O SICAR exige captcha por município — download deve ser feito manualmente. Não é possível automatizar essa etapa.
